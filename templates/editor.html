Outliner Controls:

  • Tab - Indent line (and all children)
  • Shift+Tab - Unindent line (and all children)
  • Alt+Up - Move item up (with all children)
  • Alt+Down - Move item down (with all children)
  • Enter - New line
  • Shift+Click - Hide/Unhide children of clicked item
  • Ctrl+Shift+Click - Collapse all / Expand all
  • Ctrl+S / Cmd+S - Save outline
  • Ctrl+Z / Cmd+Z - Undo
  • Ctrl+Y / Cmd+Shift+Z - Redo
